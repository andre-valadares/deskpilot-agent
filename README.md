# DeskPilot Agent

Parte do [DeskPilot](https://wol.deskpilot.xyz) — não funciona de forma independente.

---

## O que é

O DeskPilot Agent é o componente que permite **desligar seu computador por voz**. Ele roda em segundo plano no seu PC e age quando você diz "Alexa, desligar o computador".

Sem ele, o DeskPilot só consegue ligar dispositivos — que é o comportamento padrão do Wake-on-LAN. O agente completa o ciclo.

---

## Instalação

O comando de instalação fica disponível no dashboard do DeskPilot, na seção **Agente Local**, após cadastrar um dispositivo. Cada dispositivo tem seu próprio comando com token único.

---

## Parte do DeskPilot

Este repositório contém apenas o agente local. Para usar, você precisa ter o DeskPilot configurado primeiro — acesse https://wol.deskpilot.xyz para começar.
